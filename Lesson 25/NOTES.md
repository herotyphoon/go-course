# Functions

- Functions are modular, reusable block of code designed to accomplish specific tasks.
- Function take input (optional), process it, and return an output (optional).
- Functions enable us to structure and organize code.
- A program is basically an orchestration of `function` calls.
- In Go, the entry point function `main` takes no parameters and returns no value.
- Functions in Go can return multiple values.

## Syntax

```go
func functionName(parameters) returnType{
  // code block
  return
}
```
