# Primitive Data Types


### What are data types?

Data Types are just a categorization of data.

## What are primitive data types?

- They are a set of basic data types from which all other data types are constructed.
- Composite data types are made from basic data types.
- Some primitive data types are:
    - Booleans
    - Strings
    - Numbers
- Zero Values are the default values assigned to a variable on initalization.
